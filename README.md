# Web
- Lazypic 홈페이지 소스코드입니다.
- 이 코드는 AWS S3 + CloudFront + Route53을 이용해서 서비스되고 있습니다.

#### Need
1. 배포를 위해서 AWS CLI 설치가 필요합니다.

```bash
$ pip install awscli
```

1. admin@lazypic.org 를 통해서 AWS 배포 S3 Write 키를 발급받아야 합니다.
