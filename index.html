Git-hours

Git-hours is a tool that helps you work from home or freelance working time.
You can check the working hours from online.
Lazypic pursues a full time telework model and collaborates with freelancers.
Calculate your development time with Log information in Git.
For more information on how to use Git-hours, please visit Github.
The calculated time is reflected in the Lazypic Estimate System.

재택근무, 프리랜서의 업무시간을 계산하기 위해 제작된 툴입니다.
Lazypic은 풀타임 재택근무 모델을 지향하고 프리랜서와 협업하고 있습니다.
Git log 정보를 이용하여 외부에서 자유롭게 오픈소스 개발을 하더라도 업무시간을 체크할 수 있습니다.
Git-hours 사용법은 Github에서 확인할 수 있습니다. 계산된 시간은 Lazypic Estimate System에 반영됩니다.
