Freelancer Support

외부에서 자유롭게 오픈소스 개발을 하더라도 업무시간을 체크할 수 있습니다.
Lazypic은 풀타임 재택근무 모델을 지향하고 여러 프리랜서와 협업하고 있습니다.
재택근무, 프리랜서를 위한 툴. Git의 Log 정보를 이용해서 개발에 투입된 시간을 계산하세요.
Git Hours 사용법은 Github에서 확인할 수 있습니다. 계산된 시간은 Estimate System에 반영됩니다.

Start-up Support

여러분의 팀과 함께하는 도메인과 이메일.

Gmail과 메일포워딩 서비스를 연동하여, 초기 스타트업 그룹메일 고정비용을 줄일 수 있습니다.

중소규모 그룹메일 전체를 월 $1.65 비용으로 운용해보세요.

Back-end Commandline Tools

Cloud와 터미널이 바로 연계되는 Back-end Commandline Tool 제작이 가능합니다.

Aws SDK와 Go 를 활용하여 의존성이 적은 터미널 명령어를 제작, 클라우드를 활용한 파이프라인을 구축할 수 있습니다.
